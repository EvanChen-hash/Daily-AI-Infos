# AGENTS.md

## Repository rules

- Before recommending tasks, inspect README, package files, tests, TODO/FIXME comments, recent git changes, and open failing checks if available.
- Prefer small tasks that can be completed in 30–90 minutes.
- Never make code changes during planning mode.
- For execution, work on one selected task at a time.
- Before finishing execution, run the relevant lint/test/build commands.
- Do not add new production dependencies without asking me.
- Always summarize:
  - files changed
  - commands run
  - tests passed/failed
  - remaining risks
